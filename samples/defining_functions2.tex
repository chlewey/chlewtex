\documentclass[twocolumn]{article}
\usepackage{chlewfunc,amsfonts}
\begin{document}
\newcommand\R{\mathbb R}
\newcommand\N{\mathbb N}
Simple function (without mapping) used inside a paragraph
\(\funcdef f\R\) as inline math.

Simple function (with mapping) used inside a paragraph
\(\funcdef f\R[var=x]\) as inline math.

Simple function as rendered in display math:
\[\funcdef f\R[var=x]\]

Simple function with definition:
\[\funcdef f\R[var=x,definition=x^2]\]

Function with alternative writing:
\[\funcdef{\textrm{exp}}\R[var=x,notation=e^x]\]

Function with alternative notation and definition:
\[\funcdef\exp\R[
    var=x,
    notation=e^x,
    definition=\lim_{n\to\infty}\left(1+\frac xn\right)^n]\]
Or forcing display math inside the definition:
\[\funcdef\exp\R[
    var=x,
    notation=e^x,
    definition=\lim_{n\to\infty}\left(1+\frac xn\right)^n,
    display]\]

Function with different domain and codomain:
\[\funcdef[domain=\N]
    {\textrm{sqrt}}\R[
      var=n,
      notation=\sqrt{n}]
\quad
  \funcdef[codomain=\R]
    {\textrm{sqrt}}\N[
      var=n,
      notation=\sqrt{n}]
\quad
  \funcdef[domain=\N,codomain=\R]
    {\textrm{sqrt}}*[
      var=n,
      notation=\sqrt{n}]\]
      
Functions with multiple variables
\[\funcdef[
    domain=A\times B,
    codomain=C]
    {f}*[variables={a,b}]\]

Unitary operator
\[\funcdef[operator]{exp}\R[
    var=x,
    definition=\lim_{n\to\infty}\left(1+\frac xn\right)^n]\]

Binary operator
\[\funcdef[binop]{+}\R[
    vara=x,
    varb=y
 ]\]
 
Using macros in definition:
\newcommand\myabs[1]{\left\|{#1}\right\|}
\newcommand\angledprod[2]{\left\langle{#1},{#2}\right\rangle}
\[\funcdef[texop,binop,
    codomain=\R]{\angledprod}{\R^2}[
    vara=x,
    varb=y
 ]
\qquad
  \funcdef[texop,
    codomain=\R]{\myabs}{\R^2}[
    var=x,
    def=\sqrt{\angledprod{x}{x}}
 ]\]
 
More complex definitions, requiring a third line:
\[\funcdef[binop,common]{+}{\R^\R}[
    vara=f,
    varb=g,
    linedef={\forall x\in\R: (f+g)(x)=f(x)+g(x)}
 ]\]

\end{document}
